# HooBank - Modern Bank App

![HooBank Screenshot](./public/hoobank-screenshot.png)

## Overview
HooBank is a sleek, modern landing page for a futuristic bank, built with **React**, **Vite**, and **Tailwind CSS**. It is designed with a strong focus on UI/UX, featuring beautiful gradients, parallax animations, and fully responsive, component-based architecture.

## ✨ Features
- **Modern UI/UX**: Clean, visually appealing modern interface.
- **Responsive Design**: Seamless experience across mobile, tablet, and desktop devices.
- **Stylish Gradients**: Eye-catching gradients across buttons, cards, and textual elements.
- **Component-Based structure**: Highly modular and reusable React components.
- **Fast Development**: Powered by Vite for lightning-fast HMR and optimized builds.

## 🚀 Tech Stack
- **React.js**
- **Vite**
- **Tailwind CSS**

## 🛠️ Getting Started

### Prerequisites
- [Node.js](https://nodejs.org/) installed on your machine.

### Installation & Execution

1. **Clone the repository**:
```bash
git clone https://github.com/saifullahmsd/hoobank.git
cd hoobank
```

2. **Install the dependencies**:
```bash
npm install
```

3. **Run the development server**:
```bash
npm run dev
```
